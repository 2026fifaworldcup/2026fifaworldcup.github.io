Correction requests

If a group, fixture, country page, squad note, player reference, or bracket slot appears outdated, please include the exact page URL and a public source for the correction. Useful sources include official tournament pages, federation announcements, match reports, or widely accessible public schedule references.

For faster review, please describe what should change, where it appears on the page, and why the new source is more current. We avoid using private documents, paywalled-only references, or unsourced social posts as the sole basis for factual changes.

Advertising, privacy, and site questions

For advertising or privacy-related questions, include “2026 World Cup site” in the issue title so the request can be separated from ordinary content corrections. This site is static and does not provide user accounts, comments, newsletters, or login features.

Please do not send confidential personal information, passwords, ticket numbers, payment details, or private travel documents through public GitHub Issues. Public issues are visible to other visitors.

Typical review topics include broken links, missing country pages, incorrect match numbers, outdated group information, unclear Korean translations, mobile layout problems, and accessibility issues such as unreadable labels or keyboard navigation trouble. We prioritize corrections that improve factual clarity for many visitors.
